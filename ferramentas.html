Interface do criador de molduras
Existem alguns arquivos tar no arquivo FTP anônimo em file://info.cern.ch/www/src que permitem que o FRAMEmaker seja usado como uma ferramenta W3. Dan Conolly, Convex. Inclui tradução HTML MIF.
Transformando HTML em TeX
Fizemos isso com o "WWW Book" para imprimi-lo. Veja o Makefile por exemplo, e os scripts html2latex.sed e sub1.sed. Escrevemos uma introdução especial, mas fora isso todo o texto era hipertexto do projeto W3.
Gerando HTML
Estes são scripts para gerar hipertexto SGML de coisas como listas de diretórios, etc. Além disso, para verificar e corrigir HTML duvidoso.
Análise de log do servidor
A análise dos logs do servidor requer, em primeiro lugar, a alteração dos números numéricos dos nós da Internet em nomes de domínio. httpd-analyse.c é um programa para fazer isso. Alimente os resultados através de awk e grep de sua escolha!
